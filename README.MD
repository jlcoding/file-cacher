# file-cacher 
### 内网图片缓存工具


## 场景
本项目主要用于内网频繁请求外网地址，如云存储，CDN之类的，本可以帮助提速
。例如：用非固定的背景，用户头像之类的画成海报。

## 支持两种获取文件的方式
- #### **Http调用模式**
    通过调用OpenApi获取

- #### **Proxy方式**
    通过设置系统Host和系统代理
